## 介绍
一个小说/漫画阅读软件

## 功能
#### 基础功能
- [x] 自定义书源
- [x] 本地TXT阅读
- [ ] 小说阅读支持自定义字体、颜色、背景、行距、段距
- [ ] 书架自定义切换列表、网格风格
- [ ] 日夜模式切换
- [ ] 自定义主题
#### 高级功能
- [ ] 一键导出书本
- [ ] 书本分组
- [ ] 订阅消息
